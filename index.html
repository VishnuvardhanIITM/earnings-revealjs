Example: Earnings Sensitivity Model (Python)


import pandas as pd

# Simplified sensitivity model
rates = [3.5, 4.0, 4.5]
base_net_income = 620_000_000  # USD

def project_net_income(rate):
    """Higher rates improve margins up to a point."""
    spread_factor = 1 + (rate - 3.5) * 0.03
    return round(base_net_income * spread_factor, 2)

for r in rates:
    print(f"Rate: {r:.1f}% → projected net income: ${project_net_income(r):,.0f}")
Explain that this is a toy model used for demos: it shows how net income changes with interest rates. In real analysis we use more complex models, but this shows stakeholders that code can be part of the presentation.

Key Financial Metric – Return on Equity (ROE)

ROE tells us how efficiently the bank is generating profit from shareholders' equity:

\[ ROE = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

Example: If Net Income = \$620M and Equity = \$6.2B:

\[ ROE = \frac{620}{6200} = 0.10 = 10\% \]

Highlight that our ROE is compared against peer banks and the internal target. Use this slide to align on whether we are over- or under-performing.
